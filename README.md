# UNI_JSP_PROJECT
대학과제로 JSP를 이용해 만든 웹 어플리케이션 (No Code)
봉사활동을 할수 있는 업체 주소를 이용하여 좌표 DB를 생성 PostGIS 와 GeoServer OpenStreet.api 를 이용하여 JSP로 표현 하였다 .
